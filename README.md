# Hepburn Architects — Next.js + Vercel

## Run locally
```bash
npm install
npm run dev
```

## Deploy
1. Upload this project to a GitHub repository.
2. In Vercel choose **Add New → Project**.
3. Import the repository.
4. Deploy with the default Next.js settings.
5. In **Settings → Domains**, add `hepburnarchitects.net` and `www.hepburnarchitects.net`.

## Calculator lead delivery

Both calculators now hide estimates until the visitor submits a name, email address and consent.

Add these environment variables in Vercel:

- `RESEND_API_KEY` — API key from Resend
- `LEAD_TO_EMAIL` — destination inbox, normally `info@hepburnarchitects.com`
- `LEAD_FROM_EMAIL` — verified sender, e.g. `Hepburn Website <website@hepburnarchitects.net>`

Until `RESEND_API_KEY` is configured, the result will remain locked because contact details cannot be delivered safely.
