# Hepburn Architects — Next.js + Vercel

## Run locally
```bash
npm install
npm run dev
```

## Deploy
1. Upload this project to a GitHub repository.
2. In Vercel choose **Add New → Project**.
3. Import the repository.
4. Deploy with the default Next.js settings.
5. In **Settings → Domains**, add `hepburnarchitects.net` and `www.hepburnarchitects.net`.

## Important before launch
- Replace the placeholder founder portrait with David’s approved photograph.
- Replace stock project photography with genuine Hepburn projects.
- Confirm contact information and add verified testimonials only.
